# Как загрузить изменения через веб-интерфейс GitHub

Если вы вносили изменения локально (например, в `page-facont.php` или любые файлы в каталоге `facont/`) и хотите перенести их в веб-версию GitHub без работы из консоли, выполните следующие шаги:

1. Откройте нужный репозиторий на GitHub и перейдите в нужную ветку.
2. Нажмите **Add file → Upload files**.
3. Перетащите изменённые файлы или выберите их через **Choose your files**. Если файл уже существует (например, `page-facont.php`), GitHub предложит его заменить — подтвердите замену.
4. Убедитесь, что файлы отображаются в нужных путях, например:
   - `page-facont.php` в корне темы.
   - Файлы стилей и скриптов в папке `facont/` (например, `facont/facont.css`, `facont/facont.js`).
5. Внизу страницы укажите понятное сообщение коммита.
6. Нажмите **Commit changes** (можно сразу в текущую ветку или создать новую ветку и Pull Request).

Так вы сможете перенести изменения в `page-facont.php` и другие файлы прямо из браузера без использования локальных команд.
